4. Base entity

Create the package:

com.parazit.panel.common.persistence

Create an abstract base class named:

BaseEntity

Requirements:

* Annotate it with @MappedSuperclass.
* Use field access consistently.
* Use UUID as the identifier type.
* The identifier must not be manually generated by controllers or services.
* Use a Hibernate-supported UUID generation strategy compatible with the project's Hibernate version.
* Include:

UUID id
Instant createdAt
Instant updatedAt

* Enable JPA auditing for createdAt and updatedAt.
* createdAt must not be updated after insertion.
* updatedAt must be updated automatically.
* Use UTC-compatible types.
* Do not add createdBy or updatedBy yet.
* Do not use Lombok @Data.
* Implement equality carefully for JPA entities.
* Avoid including mutable fields in equals and hashCode.
* Document the chosen equals and hashCode strategy.
