6. Auto Login

Create:

XuiAuthenticationManager

Responsibilities:

Ensure authenticated session exists.

Algorithm:

If session valid

return

Else

login

If login succeeds

continue

If login fails

throw XuiAuthenticationException

Every future API request must call:

ensureAuthenticated()

before execution.

Prevent multiple concurrent logins.

Requirements:

If 20 threads require login simultaneously:

Exactly one performs login.

Others wait.

Reuse same session.

Use:

ReentrantLock

or equivalent.

Avoid synchronized on large methods.

9. Request execution

Create:

AuthenticatedRequestExecutor
