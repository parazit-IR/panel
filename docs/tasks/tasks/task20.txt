15. Snapshot integrity suite

This is a critical quality gate.

Scenario:

1. Create and activate Plan A with:

   * name A1
   * price P1
   * duration D1
   * traffic T1
   * maxDevices M1
2. Select Plan A.
3. Update Plan A to:

   * name A2
   * price P2
   * duration D2
   * traffic T2
   * maxDevices M2
4. Reload the selection.
5. Verify selection still contains A1/P1/D1/T1/M1.
6. Verify catalog/admin lookup shows A2/P2/D2/T2/M2.
7. Clear the old selection.
8. Select Plan A again.
9. Verify the new selection contains A2/P2/D2/T2/M2.

Also test:

* changing Plan status after selection does not alter the old snapshot;
* archived Plan selection remains readable until expired or cleared;
* no current Plan lookup is required to render the existing selection.
