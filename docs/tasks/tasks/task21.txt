password

connectTimeout

readTimeout

retryDelay

username: admin

# 7. Retry policy

Create:

XuiRetryExecutor

Responsibilities:

Retry transient failures.

Retry:

* timeout
* connection refused
* 502
* 503
* 504

Do NOT retry:

400

401

403

404

Validation failures

Use Spring Retry only if already present.

Otherwise implement a lightweight retry utility.
