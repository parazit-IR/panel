# Telegram Gift Codes

Gift code entry is available from the Wallet page.

```mermaid
flowchart TD
  A[Wallet] --> B[Redeem gift code]
  B --> C[Ask for code]
  C --> D[Redeem use case]
  D --> E[Wallet ledger credit]
  E --> F[Show balance/history]
```

No payment is created for gift codes. The wallet balance changes only through the wallet credit use case and immutable ledger.
